# 🤖 Cyborg Landing Page

A futuristic, responsive landing page inspired by cyberpunk aesthetics, featuring glowing UI elements, smooth animations, and an immersive user experience.

## ✨ Features

- ⚡ Fully responsive design
- 🎨 Modern cyberpunk-inspired UI
- 💡 Neon glow effects
- 🖱️ Interactive hover animations
- 📱 Mobile, tablet, and desktop compatible
- 🚀 Smooth scrolling and transitions
- 🎯 Clean and organized code structure

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript

## 🚀 Getting Started

1. Clone the repository

```bash
git clone https://github.com/your-username/cyborg-landing-page.git
```

2. Open the project folder.

3. Run `index.html` in your browser.
